### dlib环境搭建

1. 下载dlib

2. 编译
   
   ```
   cd examples
   mkdir build
   cd build
   cmake ..
   cmake --build . --config Release
   ```
   
   

### 使用

1. 用Qt打开工程

2. 将ｍodels和图片test.jpg,test1.jpg放入调试目录
